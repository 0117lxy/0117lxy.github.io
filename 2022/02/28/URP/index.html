Unity 3D project Update To URP project

  • Window->Package Manager->select ‘unity Registry’->search universal->install Universal RP

  • Assets->Create->Rendering->URP->Pipeline Assets(FR)

  • Edit->Project Settings

    • Graphics->select Universal Render Pipeline Asset
    • Quality->select Universal Render Pipeline Asset

  • Edit->Render Pipeline->Universal Render Pipeline->Update Project Materials to Universal RP Materials
